9.11数据的备份

mysqldump 是 MySQL 提供的一个命令行工具，可以生成数据库的逻辑备份。可以备份单个或多个数据库

备份单个数据库的命令：

dbname是数据库，后面是表名。表名用空格隔开，如果不指定表名就备份整个表

备份多个数据库

database参数后面 至少指定一个数据库。如果有多个用空格隔开

备份所有数据库

如果使用了 –all-databases 参数备份了所有数据库，在还原时，就不需要创建数据库，并指定要操作的数据库了

备份文件包含了use语句 create database

数据库备份文件包含了mysqldump的版本号和mysql的版本号 主机名称和备份的数据库名称等注释信息

–开头sql语言的注释

/*! */结尾都是可执行的mysql语句

source还原

source 命令是 MySQL 提供的一种用于执行 SQL 脚本的命令，可以用于还原数据库的备份。使用 source 命令可以从指定的 SQL 文件中逐条执行 SQL 语句。

  • mysql> source /path/to/backup_file.sql;
